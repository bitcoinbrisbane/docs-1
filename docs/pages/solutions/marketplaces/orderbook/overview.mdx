# Sequence Market

The Sequence Marketplace API service offers a simple and fast way to interact with sequence marketplace protocols to build a custom marketplace while accelerating your development time. The API is designed to be easy to use and to have all the features you need to build a fully functional marketplace.

---

<Callout type="tip">
Sequence Market contract is fully audited and can be found at https://github.com/0xsequence/marketplace-contracts

The officially maintained contract address is `0xB537a160472183f2150d42EB1c3DD6684A55f74c`, and is available with the same address on all networks supported by the sequence stack.

</Callout>
