# Marketplace API

:::note
[Marketplace API documentation and testing](https://0xsequence.redoc.ly/tag/marketplace)
:::

The Sequence Marketplace API service offers a simple and fast way to interact with sequence marketplace protocols. The API is designed to be easy to use and to have all the features you need to build a fully functional marketplace. Check out our various [examples](/api/marketplace/examples/orderbook-transactions) or our [custom marketplace guide](/guides/custom-marketplace) for an end to end walkthrough.
